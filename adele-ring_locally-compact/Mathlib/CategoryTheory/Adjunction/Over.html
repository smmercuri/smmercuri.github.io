    Adjunctions related to the over category #

    In a category with pullbacks, for any morphism f : X ‚ü∂ Y, the functor Over.map f : Over X ‚•§ Over Y has a right adjoint Over.pullback f.

    In a category with binary products, for any object X the functor Over.forget X : Over X ‚•§ C has a right adjoint Over.star X.

    Main declarations #

    • Over.pullback f : Over Y ‚•§ Over X is the functor induced by a morphism f : X ‚ü∂ Y.
    • Over.mapPullbackAdj is the adjunction Over.map f ‚ä£ Over.pullback f.
    • star : C ‚•§ Over X is the functor induced by an object X.
    • forgetAdjStar is the adjunction forget X ‚ä£ star X.
