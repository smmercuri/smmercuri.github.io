    Construction for the small object argument #

    Given a family of morphisms f i : A i ‚ü∂ B i in a category C and an object S : C, we define a functor SmallObject.functor f S : Over S ‚•§ Over S which sends an object given by œÄX : X ‚ü∂ S to the pushout functorObj f œÄX:

    ‚àê functorObjSrcFamily f œÄX ‚ü∂       X

            |                      |
            |                      |
            v                      v

‚àê functorObjTgtFamily f œÄX ‚ü∂ functorObj f S œÄX

    where the morphism on the left is a coproduct (of copies of maps f i) indexed by a type FunctorObjIndex f œÄX which parametrizes the diagrams of the form

    A i ‚ü∂ X
 |    |
 |    |
 v    v
B i ‚ü∂ S

    The morphism ŒπFunctorObj f S œÄX : X ‚ü∂ functorObj f œÄX is part of a natural transformation SmallObject.Œµ f S : ùü≠ (Over S) ‚ü∂ functor f S. The main idea in this construction is that for any commutative square as above, there may not exist a lifting B i ‚ü∂ X, but the construction provides a tautological morphism B i ‚ü∂ functorObj f œÄX (see SmallObject.ŒπFunctorObj_extension).
